## Supabase 테이블
- `commodity_research_cache` - 원자재 리서치
- `liquidity_cache` - 유동성 지표
- `legend_screener_cache` - 대가 스크리너 캐시 (TTL 24h)

## 완료된 기능들

### /ideas - 스크리너
- **FOMO 탭**: KR only, 필터 [ALL][VOLUME SPIKE][BREAKOUT][MOMO]
- **대가 스크리너 탭**: US only, 버튼 [버핏][오닐][피터린치][드러켄밀러][미너비니]
  - API Route: src/app/api/legend-screener/route.ts
  - FMP /stock-screener → 상위 20개만 상세조회
  - 버핏: ROE>15%, debtToEquity<0.8, netProfitMargin>10%
  - 오닐: EPS QoQ +25%, 52주 신고가 10% 이내
  - 피터린치: PEG<1, debtToEquity<0.5, epsGrowth>10%
  - 드러켄밀러: 신고가 15% 이내, revenueGrowth>20%
  - 미너비니: Yahoo Finance (VCP 패턴)
- **섹터 로테이션 탭**: 기존 유지

### /ai-trading - AI 추세추종 분석
- 차트 이미지 업로드 → Claude Vision 분석
- 4단계: 추세구조→거래량→지지저항→진입판단(YES/NO/WAIT)
- 한국어, 500자 이내
- 분석 기준 안내 드롭다운 (기본 접힘)

### /liquidity-nav - 유동성
- LIQUIDITY CYCLE 타임라인 섹션 제거됨
- LIQUIDITY FRAMEWORK 드롭다운 추가 (기본 접힘)

### /news-run - 뉴스런 PRO
**대상 채널 5개:**
- 범고래 (@bumgore)
- Aether Japan Research
- apt2me (@apt2me)
- PEF News
- Decoded Narratives ← 최근 추가

**섹션 3개:**
- 외국계 증권사 리포트 (Goldman, JPM, MS, BofA, Citi, HSBC, UBS, Nomura, Macquarie 등)
- 부동산 동향 (apt2me)
- 오늘의 핵심 이슈

**UX 방식:**
- 운영자만 브리핑 생성 (유저는 생성 불가)
- 유저는 아카이브(과거 생성된 리포트)만 열람
- 레이아웃: 좌측 사이드바(날짜 아카이브) + 우측 메인(리포트 내용)
- 기본: 가장 최근 날짜 자동 로드

**API:**
- GET /api/news-run?date=YYYY-MM-DD → Supabase 읽기만
- GET /api/news-run/dates → 존재하는 날짜 목록
- POST는 운영자 전용
